# 配置 Word 编号规则指南

## 问题说明

目前导出的 Word 文档中，标题编号（一、二、三... 或 1, 2, 3...）是纯文本，不是 Word 的内置编号系统。

要使用 Word 的编号规则，需要手动配置一次 `reference.docx` 模板，之后所有导出的文档都会自动应用编号。

## 配置步骤

### 1. 打开模板文件

打开文件：`.claude/skills/req-doc/assets/reference.docx`

### 2. 配置标题 1 的编号

1. 在文档中输入一行文字，选中它
2. 点击"开始"选项卡 → "样式"组 → 应用"标题 1"样式
3. 右键点击"标题 1"样式 → 选择"修改"
4. 在"修改样式"对话框中，点击左下角"格式"按钮 → 选择"编号"
5. 选择编号样式：
   - **阿拉伯数字**：选择 `1, 2, 3...` 格式
   - **中文数字**：选择 `一、二、三...` 格式（如果有）
6. 点击"确定"保存

### 3. 配置标题 2 的编号

1. 在文档中输入一行文字，选中它
2. 应用"标题 2"样式
3. 右键点击"标题 2"样式 → 选择"修改"
4. 点击"格式" → "编号"
5. 选择多级列表编号：`1.1, 1.2, 1.3...` 格式
6. 点击"确定"保存

### 4. 配置标题 3 的编号

1. 应用"标题 3"样式
2. 右键点击"标题 3"样式 → 选择"修改"
3. 点击"格式" → "编号"
4. 选择多级列表编号：`1.1.1, 1.1.2...` 格式
5. 点击"确定"保存

### 5. 配置标题 4 的编号

1. 应用"标题 4"样式
2. 右键点击"标题 4"样式 → 选择"修改"
3. 点击"格式" → "编号"
4. 选择多级列表编号：`1.1.1.1, 1.1.1.2...` 格式
5. 点击"确定"保存

### 6. 保存模板

1. 删除文档中的测试文字
2. 保存文件：`Ctrl+S` (Windows) 或 `Cmd+S` (Mac)
3. 关闭文件

## 使用配置好的模板

配置完成后，使用以下命令导出文档：

```bash
bash .claude/skills/req-doc/scripts/export-word-numbered.sh 安全监督风险智能分析项目需求说明书.md
```

导出的文档中，标题将自动使用 Word 的编号规则。

## 注意事项

1. **只需配置一次**：配置好的 `reference.docx` 会被所有后续导出使用
2. **编号格式可调整**：在 Word 中可以随时调整编号格式（右键标题 → 编号 → 调整编号值）
3. **中文编号**：如果 Word 没有内置中文编号格式，可以选择阿拉伯数字，导出后在 Word 中手动调整

## 验证配置

导出文档后，在 Word 中验证：
1. 右键点击标题
2. 查看是否有"调整编号值"选项
3. 如果有，说明编号配置成功
4. 如果没有，说明还是纯文本，需要重新配置模板
